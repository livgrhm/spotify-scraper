# Spotify Scraper

## Installation

1. Clone/download repo
2. Run `npm install`
3. Add Spotify client key &amp; secret in new json file in `config/env/env.json`
```json
{
    "client_id": "YOUR_KEY_HERE",
    "client_secret": "YOUR_SECRET_HERE"
}
```
4. Setup environment

## Running the app

1. Run `node app.js`
